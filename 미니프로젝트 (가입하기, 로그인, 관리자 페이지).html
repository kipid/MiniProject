# 미니프로젝트 (가입하기, 로그인, 관리자 페이지) [OO 온라인 서비스] 미니프로젝트 내용. 사용.
  1. 사용자 등록 페이지 : id, password, name, email을 접수 받아 DB에 저장
  2. 사용자 로그인 페이지 : 로그인 정보를 사용자로부터 받아서 성공/실패 여부 알림
  3. 서비스 관리자 페이지 :
        1) 등록된 사용자 목록 보여줌
        2) 특정 사용자 password 변경 저장할 수 있도록 함
